# aksharakandimalla.github.io

Personal GitHub Pages site — [aksharakandimalla.github.io](https://aksharakandimalla.github.io).

## Edit your page

- **Name, tagline, bio** — Edit `index.html` (hero section).
- **Links** — Update the `href` values in the `.links` section (GitHub, LinkedIn, email). Replace `your.email@example.com` with your real email.
- **Styling** — Tweak colors and spacing in `styles.css` (CSS variables at the top).

## Deploy

1. Commit and push to the `main` (or `master`) branch of this repo.
2. GitHub Pages will serve the site from the root. It can take a few minutes to update.
3. If needed: **Settings → Pages** → set source to your default branch and `/ (root)`.

No build step required — plain HTML and CSS.
